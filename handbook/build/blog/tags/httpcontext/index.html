HttpContext 重大调整#

在 ASP.NET 的时代，我们通常通过 HttpContext 全局静态类获取请求上下文，但在 ASP.NET Core 中，HttpContext 是一个非静态的抽象类，无法手动创建，也无法通过静态获取。

虽然在 ASP.NET Core 中无法直接获取 HttpContext 对象。但是微软也提供了注入 IHttpContextAccessor 方式获取。
