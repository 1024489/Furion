一个故意抹黑竞争对手、公开辱骂他人并大肆博同情关注的无良开源作者。一个把国外开源项目 csredis 贡献记录抹掉并占为己有的失德开发者，恶行性质和 DotNetCore.NPOI 一致。

无名小卒刘冰心（Rwing）：

占着博客园十多年博龄的无名小卒，从最开始的开源中国博客无脑恶意评论开始，进而在博客园、知乎、各大.NET群发布各种恶意评论、恶意P图并求转发的跳梁小丑。

以下（或其余）上不来台面就不过多介绍：

10. JSON 静态类

版本说明

以下内容仅限 Furion 1.16.0 + 版本使用。

10.1 获取序列化提供器#

var serializer = JSON.GetJsonSerializer();

10.2 序列化#

var str = JSON.Serialize(obj, [options]);

10.3 反序列化#

var obj = JSON.Deserialize<T>(str, [options]);

10.4 获取全局配置#

var options = JSON.GetSerializerOptions<TOption>();
