## 7.1 序列化 

```cs
var str = JsonSerializerUtility.Serialize(obj, [options]);
```

## 7.3 反序列化 

```cs
var obj = JsonSerializerUtility.Deserialize<T>(str, [options]);
```

## 7.4 获取默认序列化配置 

```cs
var options = JsonSerializerUtility.GetDefaultJsonSerializerOptionss);
```
