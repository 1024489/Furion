:::tip 服务背景

随着 Furion 框架在 NuGet 平台下载量突破 [1000 万](https://www.nuget.org/profiles/monk.soul)，越来越多的企业和个人选择将 Furion 作为项目开发的首选框架。然而，随着用户数量的急剧增加，也带来了更多的使用问题。**尽管 Furion 提供了详细的文档和 Gitee 交流社区，但我们的技术团队面临着前所未有的压力。一方面，用户咨询问题的压力日益增加，另一方面，我们需要投入更多的研发资源和时间成本。**

由于 Furion 框架的受众主要是程序员和软件企业用户，项目工期往往紧迫。为了赶进度，加班到很晚是常见的情况。**然而，在使用 Furion 框架时遇到无法解决的问题可能会对项目进度和个人绩效评估产生负面影响，甚至可能导致项目工期延误甚至法律风险。**

尽管 Furion 框架提供了在线问题反馈平台，我们也力求在当天解决出现的 Bug 问题，但面对如此庞大的开发需求和企业规定的工期，仍然无法满足需求。

**因此，我们团队决定提供专业的企业服务，成立专业团队提供一对一的支持，并承诺以最短时间给出响应。这将极大地缩短企业的开发时间和试错成本。**

:::

# 企业服务说明

感谢您选择我们基于 MIT 开源协议的框架，并对我们提供的企业服务表达兴趣。我们致力于为您的企业提供全面的技术支持和解决方案，以帮助您充分发挥我们框架的潜力。以下是我们的企业服务内容：

## 技术支持服务

- 我们的技术支持团队将提供优先级别的响应和解答，确保您在使用我们的框架时遇到的问题能够及时得到回复。
- 您可以通过电话、电子邮件或在线聊天与我们的技术支持团队联系，并享受我们专业团队的一对一支持。

## 技术培训服务

- 我们提供定制的技术培训课程，帮助您的企业团队深入了解和掌握我们框架的各项功能和最佳实践。
- 培训课程内容将根据您的需求进行调整，并可针对开发人员、管理员和其他相关人员提供不同层次和专业化的培训。

## Bug 修复服务

- 如果您在使用我们的框架时发现任何 bug 或其他技术问题，我们承诺在最短时间内提供响应，并进行相应的修复。
- 您可以通过我们的 bug 报告系统向我们提交问题，并跟踪修复进度。

## 版本更新与升级

- 我们会定期发布框架的新版本和更新，以修复漏洞、增加新功能和改进性能。
- 作为企业服务客户，您将享有提前获取最新版本和更新的特权，以确保您始终使用框架的最新功能和改进。

## 定制化开发支持

- 如果您的企业需要特定的定制化功能或开发需求，我们的专业团队可以提供相关支持，帮助您满足企业特定的业务需求。

请注意，我们的企业服务费用将根据您的具体需求、服务级别和合同约定进行商议。我们非常愿意与您进一步讨论并提供详细的报价。

对于我们的企业服务，我们承诺为您提供高质量的技术支持和完善的解决方案。如果您对以上说明有任何问题或需要进一步了解，请随时与我们联系。
